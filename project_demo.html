Project Demos

Google Places API

Android Application : Travelify

  • App uses Google's places API to search for Restaurants, Gas Stations and Hospitals in the Vicinity
  • Places are searched within a set radius of three to five miles around the user's location
  • Clicking on any place marker will open a new screen showing the detalis of that place

Google Maps Direction Finder API

Android Application : Travelify

  • Enter the destination of your travel in the text field at the top of the screen
  • User's Current location is used as the start address
  • Google Maps direction finder return a list of waypoint, I parsed through the returned list to carve out the direction
  • The app makes suggestions about" Hot Spots "(popular areas along the route)
  • Click on any Hot Spot to open a new screen which displays top rated user uploaded images at that spot

Clarifai Image Recognition API

Android Application : Travelify

  • Every User profile is maintained in the Database (MySQL)
  • User's have access to images uploaded by them and images in which they were tagged by other users
  • User can upload images by accessing camera or gallery
  • We use Clarifai API to implement #TAGRecommender, users can select appropriate tags for the image from the list
